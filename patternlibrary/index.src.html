Making a basic page, two options

To make a PlanScore page, you've got two options:

  • If you're into JavaScript and Node-based build systems, follow the instructions over in the static-site branch of the PlanScore Github repo.
  • Or roll your own by saving the source from this bare-bones HTML page.

And here are the basic styles and pieces from PlanScore.org for you to reuse.
